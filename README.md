RaspiRobot Board
================

[![RaspiRobot Board](https://dlnmh9ip6v2uc.cloudfront.net/images/products/1/1/5/6/1/11561-04_medium.jpg)  
*RaspiRobot Board (KIT-11561)*](https://www.sparkfun.com/products/11561)

The RaspiRobot Board is an expansion board for the Raspberry Pi that turns it into a robot controller. This repository contains EAGLE hardware files for the RaspiRobot Board.

Documentation for the RaspiRobot board can be found at [http://www.raspirobot.com/](http://www.raspirobot.com/).
